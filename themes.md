## 18 тем для занятий по учебной практике

1. Установка Python 3.8, pip, виртуального окружения.
2. Создание структуры папки варианта (S<n>/).
3. Проектирование ER-диаграммы в Mermaid (одна таблица, 3НФ).
4. Описание API в doc.md (CRUD + список).
5. Создание models.py: модели Peewee, поле is_active.
6. Реализация init_db() и точки входа для создания таблиц.
7. Настройка зависимостей в requirements.txt (peewee, fastapi, uvicorn, requests).
8. Реализация REST API на FastAPI в service.py (эндпоинты CRUD).
9. Валидация данных через Pydantic (схемы, Field, типы).
10. Логическое удаление (is_active=False) в DELETE-эндпоинте.
11. Фильтрация списка сущностей через query-параметры.
12. Подключение клиента к API через requests.
13. Создание desktop-приложения на tkinter (главное окно).
14. Реализация формы создания и редактирования сущности.
15. Реализация вывода списка сущностей (таблица или список).
16. Реализация удаления сущности по ID с подтверждением.
17. Обработка ошибок API (сообщения messagebox).
18. Интеграционный запуск: service.py + client.py, проверка всех эндпоинтов.

## 9 тем для самостоятельной работы

1. Доведение ER-диаграммы до 3НФ, добавление связей.
2. Реализация всех методов CRUD в models.py (без бизнес-логики).
3. Написание полной документации API в doc.md (таблицы параметров).
4. Добавление валидации на бизнес-правила (существование ID).
5. Реализация мягкого удаления с проверкой is_active.
6. Разработка всех эндпоинтов FastAPI (POST, GET, PUT, DELETE, GET list).
7. Тестирование API через curl/Postman/браузер.
8. Полное клиентское приложение на tkinter с вызовом всех эндпоинтов.
9. Отладка обработки ошибок и сообщений пользователю.
